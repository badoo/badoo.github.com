Badoo PEAR Channel

Registering the channel: 
pear channel-discover badoo.github.com
Listing available packages: 
pear remote-list -c badoo
Installing a package: 
pear install badoo/package_name
Installing a specific version/stability: 
pear install badoo/package_name-1.0.0
pear install badoo/package_name-beta
Receiving updates via a feed: 
http://badoo.github.com/feed.xml

PHPUnit_TestListener_TeamCity - PHPUnit & TeamCity Integration

Install commandpear install badoo/PHPUnit_TestListener_TeamCity
